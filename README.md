## Model模型设计

1. BasicHabit: 习惯的基础信息，创建习惯的时候生成
2. HabitRecord: 习惯记录，打卡的时候生成
3. NumberHabitRecord: 数值类型打卡记录，打卡的时候生成，打卡必填value
4. HabitIcon: 习惯的icon
5. User: 用户
